Input per testi da tradurre

Semplici input tui-input.
Toggle "Mostra traduzioni" andrà a mostrare tutti le altre lingue sotto.
Modificare il valore del toggle modificherà un impostazione per quell'utente, che da quel momento vedrà tutte le traduzioni.
Possibilità di mostrare e nascondere questo toggle: Se si è in una grande form, conviene avere la toggle esterna agli input per evitare rindondanza.
